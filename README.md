
This repository contains two end-to-end **Deep Learning** projects demonstrating the practical applications of **Artificial Neural Networks (ANN)** and **Convolutional Neural Networks (CNN)**.

## 📂 Projects Included

* 🏦 Bank Customer Churn Prediction using ANN
* 🚗 Road Accident Detection using CNN

Both projects showcase how deep learning can solve real-world classification problems using structured data and image data.

---

# 🏦 1. Bank Customer Churn Prediction using ANN

## 📌 Project Overview

Customer churn prediction helps banks identify customers who are likely to leave their services. By predicting churn in advance, banks can take proactive measures to improve customer retention and reduce revenue loss.

This project builds an Artificial Neural Network (ANN) to classify whether a customer is likely to stay or leave the bank.

### 🎯 Objectives

* Predict customer churn
* Analyze important customer attributes
* Build an Artificial Neural Network model
* Evaluate classification performance
* Improve customer retention strategies

### 🧠 Problem Type

**Supervised Machine Learning – Binary Classification**

### Features Used

* Credit Score
* Geography
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Credit Card Status
* Active Member Status
* Estimated Salary

---

# 🚗 2. Road Accident Detection using CNN

## 📌 Project Overview

Road accident detection is an important computer vision application that helps identify accidents automatically from images.

This project uses a Convolutional Neural Network (CNN) to classify accident and non-accident images with high accuracy.

### 🎯 Objectives

* Detect road accidents automatically
* Build an image classification model
* Learn visual patterns from accident images
* Improve road safety applications

### 🧠 Problem Type

**Deep Learning – Image Classification**

### Dataset

The model is trained using labeled road accident image datasets consisting of accident and non-accident images.
